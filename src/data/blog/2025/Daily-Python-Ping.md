## July 25, 2025
Write a python script to reverse a string.
```python
# Function to reverse a string
def reverse_string(s):
    return s[::-1]
```
Here is the example usage:
```python
print(reverse_string("Hello, World!"))
```
This will output:
```
!dlroW ,olleH
```
